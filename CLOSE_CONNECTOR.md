# Close Connector

## Overview

The Close Connector is a Model Context Protocol (MCP) server that integrates [glif.app](https://glif.app) AI workflows directly into language models like Claude. It enables you to run AI workflows, access multimedia generation features, and discover new workflows without leaving your LLM interface.

## Repository

**Glif MCP Server**
- Repository: [glifxyz/glif-mcp-server](https://github.com/glifxyz/glif-mcp-server)
- NPM Package: [@glifxyz/glif-mcp-server](https://www.npmjs.com/package/@glifxyz/glif-mcp-server)

## What It Does

The glif-mcp-server easily runs glif.app AI workflows inside your LLM, featuring:
- **Image Generators** - AI-powered image generation
- **Meme Creation** - Automated meme generation
- **Selfie Generation** - AI-generated selfies
- **Multimedia AI Models** - Support for all major AI models on the Glif platform

## Key Features

### Core Capabilities
- **Execute Workflows** - Run glif workflows with custom parameters
- **Query Workflows** - Retrieve detailed information about workflows and runs
- **Discover Workflows** - Search and browse featured or curated workflows
- **Account Management** - Manage user accounts and workflow collections
- **Load AI Agents** - Optional agent tools for interacting with pre-built AI agents

### Use Cases
- Leverage glif's pre-built AI workflows programmatically
- Access multimedia generation features without leaving Claude
- Discover new AI workflows and integrate them into your projects
- Build AI applications combining Claude's capabilities with specialized glif workflows
- Automate workflow execution with custom inputs

## Installation & Setup

1. Get an API token from [glif.app](https://glif.app)
2. Install via npm: `npm install @glifxyz/glif-mcp-server`
3. Configure with your glif API token
4. Connect to MCP-compatible clients (Claude Desktop, Cursor, etc.)
5. Optionally pre-load specific workflow IDs as tools for frequent use

For detailed setup instructions, refer to the [official Glif MCP Server repository](https://github.com/glifxyz/glif-mcp-server).

## Integration Benefits

This connector allows seamless communication with Glif services through the Model Context Protocol standard, enabling:
- Direct access to AI workflows from your LLM
- Programmatic control of multimedia generation
- Enhanced Claude capabilities with specialized AI tools
- Workflow discovery and management within your chat interface
